# fluentcart-mcp

MCP server for the [FluentCart](https://fluentcart.com) REST API.

> **Work in progress** — full release coming soon.

Provides ~200 AI-agent-friendly tools for managing a FluentCart store: orders, products, customers, subscriptions, coupons, reports, and more.

## Quick Start

```bash
npx fluentcart-mcp setup
```

## Authentication

Uses WordPress Application Passwords (built into WordPress 5.6+). No additional plugins required.
