## Requesting features

[Open a GitHub Issue](https://github.com/vcode-sh/fchub-plugins/issues) with:
- What you want
- Why you want it
- How often you'd use it

## What gets built

- Features multiple people request
- Features that solve real problems
- Features that don't require rewriting everything
- Features that don't break existing functionality

## What doesn't

- "Make it like YouTube" (then use YouTube)
- Features only one person wants
- Features that violate provider terms
- Features requiring server-side video hosting (that's the entire point of NOT doing this)
