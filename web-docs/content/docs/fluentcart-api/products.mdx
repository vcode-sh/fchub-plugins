title: Products
description: Full product lifecycle management. Pricing plans, variations, downloads, and product metadata.

The full product toolkit. Create, update, duplicate, and delete products — including variations, pricing, downloads, bundles, taxonomy terms, upgrade paths, and integrations. It's a lot, because products are complicated.

## Endpoints

### Core

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | [List products](/docs/fluentcart-api/listProducts) | Paginated list with filtering and search |
| `GET` | [List public products](/docs/fluentcart-api/listPublicProducts) | Public storefront products (no auth required) |
| `GET` | [Get product details](/docs/fluentcart-api/getProduct) | Detailed info for a specific product |
| `GET` | [Fetch products by IDs](/docs/fluentcart-api/fetchProductsByIds) | Retrieve multiple products by ID |
| `GET` | [Search product by name](/docs/fluentcart-api/searchProductByName) | Search products by name |
| `GET` | [Get related products](/docs/fluentcart-api/getRelatedProducts) | Related products for a given product |
| `GET` | [Suggest SKU](/docs/fluentcart-api/suggestSku) | Generate a suggested SKU |
| `GET` | [Get max excerpt word count](/docs/fluentcart-api/getMaxExcerptWordCount) | Maximum allowed excerpt length |
| `POST` | [Create product](/docs/fluentcart-api/createProduct) | Create a new product |
| `POST` | [Update product detail](/docs/fluentcart-api/updateProductDetail) | Update a product record |
| `POST` | [Duplicate product](/docs/fluentcart-api/duplicateProduct) | Duplicate an existing product |
| `POST` | [Create dummy products](/docs/fluentcart-api/createDummyProducts) | Create test products |
| `POST` | [Bulk product actions](/docs/fluentcart-api/productsBulkAction) | Perform bulk actions on products |
| `POST` | [Update editor mode](/docs/fluentcart-api/updateLongDescEditorMode) | Switch long description editor mode |
| `DELETE` | [Delete product](/docs/fluentcart-api/deleteProduct) | Delete a product |
