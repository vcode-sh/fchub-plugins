Upload, list, and manage files. Whether it's digital product deliverables or general media, these endpoints handle the storage side of things.

## Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | [List files](/docs/fluentcart-api/listFiles) | Paginated list of uploaded files |
| `GET` | [Get bucket list](/docs/fluentcart-api/getBucketList) | Available storage buckets and folders |
| `POST` | [Upload file](/docs/fluentcart-api/uploadFile) | Upload a new file via multipart form data |
| `DELETE` | [Delete file](/docs/fluentcart-api/deleteFile) | Delete an uploaded file |
