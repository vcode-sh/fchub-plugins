---
title: Settings
description: Store configuration, payment gateways, email settings, and general preferences.
---

Everything that lives in the settings panel. Store configuration, payment method management, module toggles, licence activation, confirmation pages, storage drivers, and plugin addons.

## Endpoints

### Store

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | [Get store settings](/docs/fluentcart-api/getStoreSettings) | Currency, address, checkout options, and page assignments |
| `GET` | [Get module settings](/docs/fluentcart-api/getModuleSettings) | Module configuration and activation states |
| `GET` | [Get confirmation shortcodes](/docs/fluentcart-api/getConfirmationShortcodes) | Available confirmation page shortcodes |
| `POST` | [Save store settings](/docs/fluentcart-api/saveStoreSettings) | Update store configuration |
| `POST` | [Save module settings](/docs/fluentcart-api/saveModuleSettings) | Update module activation states |
| `POST` | [Save confirmation settings](/docs/fluentcart-api/saveConfirmationSettings) | Save confirmation page settings |

### Payment Methods

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | [Get all payment methods](/docs/fluentcart-api/getAllPaymentMethods) | All configured payment methods |
| `GET` | [Get payment method settings](/docs/fluentcart-api/getPaymentMethodSettings) | Settings for a specific method |
| `GET` | [Get connect info](/docs/fluentcart-api/getPaymentMethodConnectInfo) | OAuth URLs and connection info |
| `POST` | [Save payment method settings](/docs/fluentcart-api/savePaymentMethodSettings) | Save method configuration |
| `POST` | [Save payment method design](/docs/fluentcart-api/savePaymentMethodDesign) | Save checkout display settings |
| `POST` | [Reorder payment methods](/docs/fluentcart-api/reorderPaymentMethods) | Change method display order |
| `POST` | [Disconnect payment method](/docs/fluentcart-api/disconnectPaymentMethod) | Disconnect a payment method |
