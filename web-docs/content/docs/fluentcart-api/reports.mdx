title: Reports
description: Revenue, order, and product analytics. Date-filtered reporting endpoints.

Numbers, charts, and analytics. Revenue breakdowns, order charts, product performance, customer insights, subscription cohorts, refund analysis, and retention metrics. All date-filterable, all JSON.

## Endpoints

### Overview

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | [Get reports overview](/docs/fluentcart-api/getReportsOverview) | Summary for the reports dashboard |
| `GET` | [Get report overview](/docs/fluentcart-api/getReportOverview) | Summarised report overview |
| `GET` | [Get dashboard summary](/docs/fluentcart-api/getDashboardSummary) | Key metrics for the main dashboard |
| `GET` | [Get report dashboard stats](/docs/fluentcart-api/getReportDashboardStats) | Dashboard statistics |
| `GET` | [Get report meta](/docs/fluentcart-api/getReportMeta) | Report metadata |
| `GET` | [Get report sources](/docs/fluentcart-api/getReportSources) | Available report data sources |
| `GET` | [Get quick order stats](/docs/fluentcart-api/getQuickOrderStats) | Quick order statistics for a date range |
| `GET` | [Get recent orders](/docs/fluentcart-api/getRecentOrders) | Most recent orders for the dashboard |
| `GET` | [Get recent activities](/docs/fluentcart-api/getRecentActivities) | Recent activity log entries |
