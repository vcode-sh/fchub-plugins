## v1.0.0 <Badge>Initial Release</Badge>

- 3 icon types: SVG, Dashicons, Image URL
- Scrollable container with auto-fit viewport and fixed height modes
- Endpoint reordering, active/inactive toggle, auto-slug generation
- Dark mode support
- Full uninstall cleanup
