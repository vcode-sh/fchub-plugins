The currency switcher is how your visitors pick their currency. Not a sad native browser dropdown — a proper custom widget with country flags, smooth animations, and a little green dot that says "yes, these rates are fresh." One shortcode, works anywhere.

## Add It to Your Site

```text
[fchub_currency_switcher]
```

Paste this shortcode into any page, post, widget area, or template file. That's the entire setup. No attributes, no parameters, no configuration needed — the widget pulls everything from your [plugin settings](/docs/fchub-multi-currency/configuration).

If the plugin is disabled or you haven't added any currencies yet, the shortcode outputs nothing. No broken markup, no empty boxes.

You can also place it multiple times on the same page — header and footer, for example. Each instance works independently.

## What Happens When Someone Switches

<Steps>
  <Step>
    **Visitor clicks a currency**

    The button dims slightly to show something's happening.
  </Step>
  <Step>
    **Preference is saved**

    Their choice is remembered — in a browser cookie (sticks for 90 days) and in their account if they're logged in. No need to pick again next time.
  </Step>
  <Step>
    **Page refreshes with new prices**

    The page reloads and every price on the site is now in the selected currency. Product cards, cart totals, checkout — everything updates.
  </Step>
</Steps>

## Rate Freshness Badge

The little footer inside the dropdown that reads "Rates updated 2 hours ago" with a coloured dot.

- **Green dot** — rates are fresh, everything's current
- **Red dot** — rates haven't been updated in a while

You can toggle this on or off in the [Checkout settings](/docs/fchub-multi-currency/configuration#checkout). It's a small touch, but it builds trust — visitors can see that the converted prices aren't based on exchange rates from three months ago.

## Keyboard Navigation

The switcher is fully keyboard-accessible — you can navigate the whole thing without touching a mouse:

| Key | What It Does |
|---|---|
| **Enter** or **Space** | Opens the dropdown, or selects the focused currency |
| **Arrow Down / Up** | Moves through the currency list |
| **Home / End** | Jumps to the first or last currency |
| **Escape** | Closes the dropdown |
| **Tab** | Closes the dropdown and moves focus along naturally |

## No JavaScript? No Problem

If a visitor has JavaScript disabled (rare, but it happens), they get a plain native dropdown as a fallback. Same currencies, same functionality, just less pretty.
