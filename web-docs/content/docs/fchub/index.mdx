FCHub is a suite of plugins built for [FluentCommunity](https://www.flavor.dev/wordpress-plugins/fluent-community/) — the WordPress community platform by WPManageNinja. Each plugin adds functionality that the community has been asking for but hasn't shipped yet.

## Why FCHub Exists

FluentCommunity is a solid community platform. But like most WordPress plugins, it ships with a core feature set and leaves the rest for "future updates." Some of those updates take a while. Sometimes a long while.

FCHub fills the gaps. Video uploads and more — features that modern communities expect out of the box. Each plugin works independently, so you only install what you need.

## How It Works

- **Separate plugins** — Install only what you need. No bloat.
- **Built for FluentCommunity** — Deep integration, not a generic bolt-on.
- **Licensed per plugin** — Each plugin has its own license. Free trials available at [fchub.co](https://fchub.co).
- **Independent updates** — Each plugin updates on its own schedule.
