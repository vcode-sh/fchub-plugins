FCHub is a collection of open-source plugins for [FluentCart](https://www.flavor.dev/wordpress-plugins/fluent-cart/) and [FluentCommunity](https://www.flavor.dev/wordpress-plugins/fluent-community/) — the WordPress e-commerce and community platforms by WPManageNinja.

Payments, invoicing, memberships, video streaming, WooCommerce migrations — things people actually need but nobody got around to building. So I did.

## How It Works

- **Separate plugins** — Install only what you need. No bloat.
- **Open source** — GPLv2. Code's on [GitHub](https://github.com/vcode-sh/fchub-plugins). Read it, fork it, improve it.
- **Independent updates** — Each plugin has its own release cycle.
- **PHP 8.3+** — Modern PHP only. Not negotiable.
- **WordPress 6.8+** — Keeps things sane.
