## v1.0.0 <Badge>Initial Release</Badge>

The first public release of FCHub Memberships — a complete membership system for FluentCart.

### Plan Management
- Create unlimited membership plans with title, slug, description, status, and level
- Plan hierarchy with `includes_plan_ids` for tiered access
- Duration types: lifetime, fixed days, subscription mirror
- Trial periods with configurable trial days
- Grace periods for cancellation and failed renewal
- Custom restriction messages and redirect URLs per plan
- Scheduled plan status changes (activate/deactivate at a future date)
- Plan duplication

### Content Protection
- Protect posts, pages, and any public custom post type
- Protect taxonomy terms (categories, tags, custom taxonomies) with inheritance
- Menu item protection (hide nav items from non-members)
- URL pattern protection with exact, prefix, and regex matching
- Comment protection
- Special page protection (blog, front page, search, 404, author, date archives)
- Per-post protection via meta box with plan selection and teaser configuration
- Bulk protect/unprotect actions in post list tables
- Teaser modes: excerpt, more tag, first N words, custom text
- Custom restriction messages with placeholder support
- CTA buttons on restriction blocks
- Archive filtering (exclude protected posts from archives/search)
- REST API content filtering
- Admin bypass (configurable)

### Drip Content
- Delay-based drip (unlock X days after grant)
- Fixed-date drip (unlock on a specific calendar date)
- Drip timeline and calendar admin views
- Email notifications when content unlocks
- Notification retry logic (3 attempts)
- Drip milestones at 25%, 50%, 75%, 100%
- Frontend drip progress bars

### Trials
- Configurable trial periods per plan
- Automatic trial-to-paid conversion when subscription payment confirmed
- Automatic trial expiration when no payment received
- Trial expiring notifications (configurable notice period)
- Dedicated trial lifecycle cron job

### Email Notifications
- 8 transactional email types: AccessGranted, AccessRevoked, AccessExpiring, DripContentUnlocked, MembershipPaused, MembershipResumed, TrialConverted, TrialExpiring
- Customizable HTML email templates with smart codes
- Responsive email wrapper
- Async delivery via Action Scheduler
- Individual enable/disable per email type

### FluentCart Integration
- `BaseIntegrationManager` integration module
- Integration feeds with plan selection
- Validity modes: lifetime, fixed duration, mirror subscription
- Grace period configuration
- Cancellation behavior: immediate or wait for validity
- Auto-create WordPress user from order email
- Access revocation on cancel/refund events
- Subscription validity watcher (5-minute cron)
- Event lock table for idempotent processing

### FluentCRM Integration
- 15 automation triggers covering lifecycle, trial, drip, and engagement events
- 7 automation actions: grant, revoke, pause, resume, extend, change plan, create coupon
- 7 benchmarks for automation flow control
- 25+ smart codes for email personalization
- Contact profile section showing membership status
- Segment filters: by status, plan, and membership duration
- Checkout URL and upgrade URL smart codes
- Payment update URL smart code

### FluentCommunity Integration
- Automatic space/group membership on plan grant/revoke
- Badge assignment based on plan-to-badge mappings
- Badge revocation on membership revoke (configurable)
- Resource search for admin plan rule editor

### LearnDash Integration
- Course and lesson protection through membership plans
- Enrollment grant/revoke through adapter

### Analytics & Reports
- Overview dashboard: active members, plans, churn rate, new/churned this month
- Members-over-time chart (30d, 6m, 12m)
- Plan distribution breakdown
- Churn analysis
- Revenue by plan
- Content popularity
- Daily stats aggregation cron
- REST API for all report data

### Admin Interface
- Single-page admin application (Vue.js + Vite)
- Plan management with content rules editor
- Member management with grant details
- Content protection rules manager
- Drip schedule calendar and notification queue
- Import tool with preview
- Settings page with email template editor
- Reports dashboard

### Frontend
- 4 shortcodes: `[fchub_restrict]`, `[fchub_membership_status]`, `[fchub_drip_progress]`, `[fchub_my_memberships]`
- 2 Gutenberg blocks: restrict, membership-status
- Account page with content library and history
- Frontend CSS with clean, overridable styles

### Developer Tools
- Full REST API (plans, members, content, drip, reports, settings, import)
- WP-CLI commands: list-grants, grant, revoke, revoke-by-order, check, backfill, sync, expire-check, drip-process, purge-expired, debug, stats, export-members
- 5 webhook events with HMAC-SHA256 signatures
- Multiple webhook URL support
- Async webhook delivery via Action Scheduler
- Adapter system for extensibility
- 10 custom database tables with foreign keys and indexes
- CSV import with generic and PMPro parsers
- Audit log with 90-day auto-cleanup
- 7 WP-Cron events + 2 Action Scheduler hooks

### Database
- 10 custom tables: plans, plan_rules, grants, grant_sources, event_locks, protection_rules, validity_log, drip_notifications, stats_daily, audit_log
- Foreign key constraints with CASCADE and SET NULL
- Optimized indexes for all query patterns
- Migration system with versioning (V1, V2, V3)
