# FCHub Memberships

Version: `1.1.0`

FCHub Memberships is a WordPress membership plugin for FluentCart with:

- plan management
- content protection
- drip scheduling
- member lifecycle management
- reporting and admin tooling

## Development

The tracked source includes local development tooling:

- PHPUnit for PHP tests
- Vite for the admin app build

## Versioning

Current plugin version: `1.1.0`
